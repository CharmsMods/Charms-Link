4. TRANSITION LOGIC
-------------------
The section uses a Split-view Canvas approach with a draggable divider.
The JS uses `data-before` and `data-after` attributes on `.example-card` elements to load images into a canvas.
The split position is controlled via mouse and touch events.
The modal replicates this behavior in fullscreen.
