HOW TO REPLICATE THE CUSTOM CURSOR ("CURSOR JACK") EFFECT

To implement the custom cursor effect seen on the page, you need three parts: the HTML element, the CSS styling (including the global cursor hide), and the JavaScript to track movement and handle hover states.

--- 1. THE HTML ---
Place this element right at the start of your <body> tag:

<div id="cursor"></div>

--- 2. THE CSS ---
Add this to your stylesheet. 
Note: This includes a 'mix-blend-mode' which makes the cursor change color based on the background for a premium look.

/* 1. Hide the default browser cursor everywhere */
* {
    cursor: none !important;
}

/* 2. Style the custom cursor element */
#cursor {
    position: fixed;
    top: 0;
    left: 0;
    width: 20px;
    height: 20px;
    border: 2px solid #d47449; /* Current accent color: --accent-rust */
    background: rgba(212, 116, 73, 0.2);
    transform: translate(-50%, -50%); /* Centers the div on the mouse coordinates */
    pointer-events: none; /* VERY IMPORTANT: Ensures clicks pass through to links/buttons */
    z-index: 9999;
    transition: width 0.1s, height 0.1s, background 0.2s; /* Smooth expansion on hover */
    mix-blend-mode: difference; /* Optional: Inverts colors against background for high visibility */
}

/* 3. Style for when the cursor is hovering over interactive elements */
#cursor.hovered {
    width: 50px;
    height: 50px;
    background: #d47449;
    border: none;
}

--- 3. THE JAVASCRIPT ---
Add this script before your closing </body> tag.

<script>
    const cursor = document.getElementById('cursor');
    
    // Track mouse movement
    document.addEventListener('mousemove', (e) => {
        cursor.style.left = e.clientX + 'px';
        cursor.style.top = e.clientY + 'px';
    });

    // Handle hover states for interactive elements
    // Add any CSS selectors you want the cursor to react to:
    const interactiveElements = 'a, button, .code-block, .sidebar li a';
    
    document.querySelectorAll(interactiveElements).forEach(el => {
        el.addEventListener('mouseenter', () => cursor.classList.add('hovered'));
        el.addEventListener('mouseleave', () => cursor.classList.remove('hovered'));
    });
</script>

--- SUMMARY OF SETTINGS ---
- Default Color: #d47449 (with 0.2 opacity background)
- Hover Color: #d47449 (solid)
- Default Size: 20px x 20px
- Hover Size: 50px x 50px
- Easing: steps(4) or smooth transition (0.1s)
- Blend Mode: difference (high contrast against text/background)
