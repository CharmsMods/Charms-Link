Digital Grain Studio Manual

Overview

This tool lets you layer customizable noise over any image, producing digital “film grain,” analog texture, or stylized distortion.

Controls

  • Noise Strength (σ): sets how strong the noise signal is — higher means more contrast and busier grain.
  • Color Noise: when on, each channel gets independent noise; when off, monochrome grain.
  • Noise Size: scales the pixelation of the noise — small values = fine grain, large = chunky texture.
  • Blurriness: applies Gaussian blur to the noise only, softening edges for film-like smoothness.
  • Blend Mode: controls how the noise mixes with your image pixels.
  • Noise Opacity: transparency of the noise layer; lower for subtle realism.

Blend Mode Reference

  • Normal: places the noise directly on top — no blending math.
  • Overlay: increases contrast; mid-tones stay stable, highlights brighten, shadows deepen. Great general choice.
  • Screen: lightens the image, as though projecting extra light.
  • Multiply: darkens, similar to stacking film transparencies.
  • Add (lighter): literally adds pixel values, producing bright energetic bloom — beware of clipping.
  • Subtract (difference): subtracts pixel values, creating high-contrast inversion effects.

Workflow Tips

  • Fine-grain film feel → Overlay, opacity ≈ 0.25, blur ≈ 2 px, noise size 3–6.
  • Dreamy haze → Screen, opacity ≈ 0.2, blur ≥ 3 px, large noise size > 20.
  • Gritty darkness → Multiply, opacity ≈ 0.3, blur ≈ 1 px.
  • Glitch art → Difference, strong color noise, opacity 0.4–0.6.
