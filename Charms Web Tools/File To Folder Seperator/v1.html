File To Folder Seperator

Flatten Folder

Select a folder to scan. The tool will copy all files (recursively) into a single new flat folder. A structure.json file will be created to map files back to their original paths. Duplicate filenames will be automatically renamed.

Reconstruct Folder

Select the folder containing the flattened files, the structure.json map, and a destination folder. The tool will rebuild the original directory structure.
